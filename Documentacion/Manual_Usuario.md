# Manual de Usuario AutoRevolution

## Acceso

Abrir el archivo `index.html` en un navegador web.

## Panel principal

Muestra el resumen de clientes, vehículos, citas y mantenimientos registrados.

## Clientes

Completar el formulario de clientes y presionar `Guardar Cliente`.

## Vehículos

Completar los datos del vehículo y presionar `Guardar Vehículo`.

## Citas

Registrar cliente, vehículo, fecha, hora y servicio solicitado.

## Mantenimiento

Registrar vehículo, fecha, descripción del servicio y costo.

## Nota

Este sistema funciona como prototipo académico. Los datos se guardan temporalmente en el navegador mediante localStorage.
